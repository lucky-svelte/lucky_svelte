# lucky_svelte

A view helper for rendering Svelte components in Lucky views

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     lucky_svelte:
       github: lucky-svelte/lucky_svelte
   ```

2. Run `shards install`

## Usage

```crystal
require "lucky_svelte"
```
